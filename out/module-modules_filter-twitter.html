(inner) groupIntoConvos(acc, message) → {Array.<Array.<module:modules/filter-twitter~Message>>}

A reduce function that accepts an accumlator array and a Message Object. It uses checkIds to see if the prop id_str of the recipient and sender are both contained within any of the conversations. If they are both found within one of the conversations groups in the accumlator, the Message object is added to that group, if not a new group is created and added to the accumlator. A conversation group is an array of Message Objects in which each object has the same 2 id_str found for either recipient or the sender Object.
