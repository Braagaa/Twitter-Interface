(inner, constant) textareaTweetHandler

Keyup event handler for posting a tweet on the textarea. Checks the character count of the text. If it is in acceptable range, the post tweet button is still enabled and background color of the count is left alone. If it is not in range, the post button is disabled and the background color of count is changed to indicate that the text is over the limit.
